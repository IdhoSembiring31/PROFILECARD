# 📱 ProfileCard Pro

Aplikasi profil dengan fitur kamera dan GPS. Dibangun dengan Expo & React Native.

---

## ✨ Fitur Utama
- 📸 Ambil Foto dari Kamera
- 📍 Deteksi Lokasi GPS
- 🆔 Tampilkan Kartu Profil
- ℹ️ Tampilan Versi Aplikasi (Bonus A)

## 🛠 Cara Install
1. Download file `.apk` dari link di atas.
2. Buka file di HP Android, izinkan instalasi dari sumber tidak dikenal.
3. Klik **Install**, tunggu selesai.
4. Buka aplikasi dari menu utama HP.
