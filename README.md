## ✨ Fitur Unggulan
1. **Menampilkan Data Profil:** Menampilkan nama, NIM, dan informasi pengguna secara dinamis.
2. **Ambil Lokasi Real-time:** Menggunakan GPS untuk menampilkan koordinat Latitude dan Longitude.
3. **Buka Google Maps:** Navigasi langsung menuju lokasi pengguna dengan satu tombol.
4. **Galeri & Hapus Foto:** Mengambil foto dari galeri dan menghapusnya dari tampilan.
