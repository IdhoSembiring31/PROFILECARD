## ✨ Fitur Unggulan
1. **Menampilkan Data Profil:** Menampilkan nama, NIM, dan informasi pengguna.
2. **Ambil Lokasi Real-time:** Menggunakan GPS untuk menampilkan koordinat.
3. **Buka Google Maps:** Navigasi langsung ke lokasi pengguna.
4. **Galeri & Hapus Foto:** Mengambil foto dari galeri dan menghapusnya.
